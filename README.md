# DOB Age Calculator - Online

## Overview

DOB Age Calculator is a simple web application that calculates a person's exact age based on their date of birth.

The application determines age in years, months, and days using JavaScript date calculations and provides results instantly through a clean and easy-to-use interface.

---

## Features

### Exact Age Calculation

Calculates age based on the selected date of birth.

### Instant Results

Displays results immediately without page refresh.

### Responsive Design

Works on desktop, tablet, and mobile devices.

### User-Friendly Interface

Simple and intuitive layout for quick calculations.

### Browser-Based

Runs directly in modern web browsers without installation.

---

## Technology Stack

* HTML
* CSS
* JavaScript

---

## How It Works

1. Select a date of birth.
2. Click the calculate button.
3. The application computes the age difference between the current date and the selected date.
4. Results are displayed instantly.

---

## Installation

1. Download or clone the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.

No additional dependencies are required.
